# Custom System Installer

This repository was generated by the `archgen` utility using a working system as a template. It contains all of your explicitly installed Pacman packages, AUR packages, dotfiles, assets, and themes.

## How to Install

On a fresh Arch Linux or Arch-based distro installation, open a terminal and run the following:

1. **Clone this repository:**
   ```bash
   git clone https://github.com/Void-Lotus/my-arch-installer.git
   cd my-arch-installer
   ```

2. **Run the installer script:**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

3. **Follow the on-screen prompts** to select which components to configure (system packages, AUR, configurations, wallpapers, and/or SDDM login screens).
